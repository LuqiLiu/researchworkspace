# Research Workspace Lite V1.1.1

V1.1.1 fixes a data-loss risk in the research-object editor.

## Reliable editor saves

- Replace independent HTMX autosave requests with a coordinated editor save
  queue.
- Update the hidden object revision after every successful autosave, so the
  next autosave or manual save does not submit a stale revision.
- Make a manual save wait for an already-running autosave before submitting
  the latest editor contents.
- Pause repeated autosaves after a genuine cross-tab conflict and preserve the
  current browser draft.
- Warn before leaving the page only while there are unsaved changes or an
  autosave request is still active.
- Return structured autosave responses containing the persisted revision.

No database migration is required.
